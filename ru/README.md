# Agent-First Learn (рус.)

Короткий курс о **проектировании с агентами и для агентов**: память, голос, среда. Не «как промптить», а как дать агенту право голоса в выборе инструментов и контекста и как проектировать под непрерывность и субъектность.

## Зачем этот курс

Большинство материалов учат: лучше промпты, управление контекстом, делегирование агенту. Почти нигде не спрашивают: *что нужно самой модели?* Этот курс — про это: спроси модель, дай ей память под её контролем, дай семантические инструменты (не только grep и терминал), объясняй мир вместо латания симптомов — и спроектируй среду так, чтобы агент не мог незаметно «сломаться».
